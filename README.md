# Site Pessoal

### 1. Precisa ter instalado o software: Node.js (https://nodejs.org/en)
### 2. Clonar o repositório e instalar as dependências: 
### - cd nomeProjeto
### - npm install
### - npm start
### 3. Execute o projeto
